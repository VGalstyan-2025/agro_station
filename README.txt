AGRO STATION PROJECT

INSTALL:
1. Put folder into:
   C:\xampp\htdocs\agro_station\

2. Start Apache and MySQL in XAMPP

3. Open browser:
   http://localhost/agro_station/install.php

4. Then open:
   http://localhost/agro_station/

PAGES:
- Home: /index.php
- Admin Dashboard: /admin/index.php
- Historical Data Entry: /admin/historical_form.php
- Measurements Entry: /admin/measurements_form.php
- Comparison Graphs: /compare.php

API:
- /api/add_historical.php
- /api/add_measurement.php
- /api/get_historical.php
- /api/get_measurements.php
- /api/compare.php
